# 蚕食单词

![preview](https://github.com/user-attachments/assets/c75f80de-0ec9-4173-9d22-9c269303594b)


## 功能
- 切换展示 初中、高中、CET4、CET6、考研、托福、SAT 词库
- 点击词条暗化它，表示已知，有点蚕食的感觉
- Ctrl + 点击，取消标记某个词条
- 右击显示词条注释
